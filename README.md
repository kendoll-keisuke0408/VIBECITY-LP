# VIBECity RECORDS Official Website

架空のAI音楽レーベル「VIBECity RECORDS」の公式ランディングページです。
AIと人間が共創する新しい音楽の形を追求し、架空都市「VIBE CITY」の世界観を表現しています。

## Features

- **Immersive Design**: ニューヨークのスタイリッシュな雰囲気とサイバーパンクな要素を融合したデザイン。
- **Music & Video Player**: 楽曲の試聴やミュージックビデオの再生が可能。
- **Responsive Layout**: PC、タブレット、スマートフォンに対応。
- **Interactive Elements**: スクロールアニメーションやホバーエフェクトによる動的な体験。

## Technologies

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- JavaScript (Vanilla JS)
- GSAP (Animation Library)
